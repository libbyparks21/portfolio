Tree Simulation and Modeling
Created in Speedtree for Houdini Layout - November 2023

Populated city with trees that matched the style, simulated them to reflect that it is a calm autumn day. And kept it LIGHTWEIGHT. Trees had to react to an explosion. Modeled and designed 8 different trees with unique "wonky" feeling. Added a calm, physically accurate leaf and branch simulation. Added them to the current layout using the USD Pipeline. Optimized layout by assuring that only the trees in focus will move and drop leaves, the rest will be static instanced copies for background feel. Created a simulation for the trees reacting to the explosion.

(SpeedTree, File-Caching, USD)
